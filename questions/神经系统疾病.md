# 神经系统疾病

## 1. 头痛
**症状**：头部疼痛，可伴有恶心、畏光
**解决办法**：
- 休息，保持安静
- 服用止痛药（布洛芬、对乙酰氨基酚）
- 冷敷或热敷
- 避免触发因素（光线、噪音）
- 严重时就医
- 规律作息

## 2. 失眠
**症状**：入睡困难、睡眠浅、易醒、早醒
**解决办法**：
- 规律作息时间
- 睡前避免咖啡因
- 适度运动
- 放松训练
- 必要时服用安眠药
- 营造良好睡眠环境
- 睡前避免手机

## 3. 颈椎病
**症状**：颈部疼痛、僵硬、上肢麻木、头晕
**解决办法**：
- 正确姿势办公
- 颈部牵引
- 物理治疗
- 服用消炎止痛药
- 适度运动
- 避免长时间低头
- 严重时手术

## 4. 腰椎间盘突出
**症状**：腰痛、下肢放射痛、麻木
**解决办法**：
- 卧床休息
- 服用止痛药和神经营养药
- 物理治疗
- 牵引治疗
- 游泳锻炼
- 严重时手术
- 避免弯腰负重

## 5. 抑郁症
**症状**：情绪低落、兴趣减退、乏力、睡眠障碍
**解决办法**：
- 抗抑郁药（SSRIs等）
- 心理治疗
- 运动疗法
- 规律作息
- 社交支持
- 严重时住院治疗

## 6. 焦虑症
**症状**：过度担忧、心慌、胸闷、出汗
**解决办法**：
- 抗焦虑药
- 心理治疗（认知行为疗法）
- 放松训练
- 运动
- 避免咖啡因
- 规律作息

## 7. 偏头痛
**症状**：一侧头部搏动性疼痛、恶心、畏光、畏声
**解决办法**：
- 避免诱因（奶酪、巧克力、酒精）
- 服用曲普坦类药物
- 预防性用药（频繁发作）
- 冷敷
- 休息于暗室
- 记录头痛日记
