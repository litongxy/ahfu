# 骨骼肌肉疾病

## 1. 关节炎
**症状**：关节疼痛、肿胀、僵硬、活动受限
**解决办法**：
- 休息
- 服用非甾体抗炎药
- 物理治疗
- 关节腔注射
- 适度运动
- 控制体重
- 严重时关节置换

## 2. 类风湿性关节炎
**症状**：对称性关节肿痛、晨僵
**解决办法**：
- 非甾体抗炎药
- 抗风湿药（甲氨蝶呤）
- 生物制剂
- 适度运动
- 定期复查

## 3. 骨质疏松
**症状**：腰背痛、骨折易发、身高缩短
**解决办法**：
- 补充钙剂和维生素D
- 适度负重运动
- 抗骨质疏松药物（双膦酸盐等）
- 预防跌倒
- 戒烟限酒
- 定期骨密度检查

## 4. 痛风
**症状**：关节红肿热痛（常为大脚趾）
**解决办法**：
- 急性期服用秋水仙碱
- 服用非甾体抗炎药
- 低嘌呤饮食
- 多喝水
- 服用降尿酸药（缓解期）
- 避免海鲜、啤酒
- 控制体重

## 5. 颈椎病
**症状**：颈部疼痛、僵硬、上肢麻木、头晕
**解决办法**：
- 正确姿势办公
- 颈部牵引
- 物理治疗
- 服用消炎止痛药
- 适度运动
- 避免长时间低头
- 严重时手术

## 6. 腰椎间盘突出
**症状**：腰痛、下肢放射痛、麻木
**解决办法**：
- 卧床休息
- 服用止痛药和神经营养药
- 物理治疗
- 牵引治疗
- 游泳锻炼
- 严重时手术
- 避免弯腰负重

## 7. 肩周炎
**症状**：肩部疼痛、活动受限
**解决办法**：
- 物理治疗
- 功能锻炼
- 口服止痛药
- 封闭治疗
- 手术治疗（粘连严重）
- 注意保暖
