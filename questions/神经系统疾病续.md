# 神经系统疾病(续)

## 1. 帕金森病
**症状**：静止性震颤、肌强直、运动迟缓
**病因**：黑质多巴胺能神经元变性
**解决办法**：
- 左旋多巴
- 多巴胺受体激动剂
- 手术治疗（DBS）
- 康复训练
- 定期复查
**预防**：无法预防

## 2. 阿尔茨海默病
**症状**：记忆力减退、定向障碍
**病因**：神经细胞变性
**解决办法**：
- 胆碱酯酶抑制剂
- 美金刚
- 认知训练
- 护理照料
- 定期复查
**预防**：多用脑

## 3. 运动神经元病

- 对症治疗
- 康复训练

**预防**：无法预防
