# 内分泌代谢疾病

## 1. 糖尿病
**症状**：多饮、多尿、多食、体重下降、乏力
**解决办法**：
- 饮食控制（低碳水化合物）
- 规律运动
- 口服降糖药或胰岛素治疗
- 定期监测血糖
- 控制体重
- 定期检查并发症
- 足部护理

## 2. 甲状腺功能亢进
**症状**：心悸、多汗、手抖、体重下降、眼球突出
**解决办法**：
- 服用抗甲状腺药物（甲巯咪唑、丙基硫氧嘧啶）
- 放射性碘治疗
- 手术治疗（部分）
- 避免碘摄入过多
- 减少压力
- 定期复查

## 3. 甲状腺功能减退
**症状**：乏力、畏寒、体重增加、便秘、记忆力减退
**解决办法**：
- 服用左甲状腺素钠
- 定期复查甲功
- 避免过量摄入碘
- 适度运动
- 注意保暖

## 4. 痛风
**症状**：关节红肿热痛（常为大脚趾）
**解决办法**：
- 急性期服用秋水仙碱
- 服用非甾体抗炎药
- 低嘌呤饮食
- 多喝水
- 服用降尿酸药（缓解期）
- 避免海鲜、啤酒
- 控制体重

## 5. 骨质疏松
**症状**：腰背痛、骨折易发、身高缩短
**解决办法**：
- 补充钙剂和维生素D
- 适度负重运动
- 抗骨质疏松药物（双膦酸盐等）
- 预防跌倒
- 戒烟限酒
- 定期骨密度检查

## 6. 高血脂
**症状**：常无症状，体检发现
**解决办法**：
- 他汀类药物
- 低脂饮食
- 适度运动
- 控制体重
- 定期复查血脂
