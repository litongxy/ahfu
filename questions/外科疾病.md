# 外科疾病

## 1. 疝气
**症状**：腹股沟肿块、可还纳
**病因**：腹壁薄弱
**解决办法**：
- 疝修补术
- 腹腔镜手术
- 避免重体力
- 定期复查
- 定期复查
**预防**：避免腹压

## 2. 阑尾炎(续)
**症状**：转移性右下腹痛
**病因**：阑尾梗阻
**解决办法**：

- 定期复查
**预防**：无法预防
