# 皮肤疾病(续)

## 1. 接触性皮炎
**症状**：红斑、水肿、水疱
**病因**：接触致敏物
**解决办法**：
- 避免接触
- 激素软膏
- 抗组胺药
- 冷敷
- 定期复查
**预防**：避免接触

## 2. 药物性皮炎
**症状**：皮疹、瘙痒、严重可剥脱
**病因**：药物过敏
**解决办法**：
- 停用致敏药
- 激素治疗
- 抗组胺药
- 支持治疗
- 避免再用
**预防**：慎用药物

## 3. 神经性皮炎
**症状**：瘙痒、苔藓样变
**病因**：精神因素
**解决办法**：
- 激素软膏
- 抗组胺药
- 心理治疗
- 避免抓挠
- 调节情绪
**预防**：调节情绪

## 4. 脂溢性皮炎
**症状**：红斑、油腻、头皮屑
**病因**：马拉色菌感染
**解决办法**：
- 抗真菌洗发水
- 激素药膏
- 钙调磷酸酶抑制剂
- 生活方式调整
- 定期复查
**预防**：清洁

## 5. 玫瑰痤疮
**症状**：面部潮红、毛细血管扩张
**病因**：血管神经功能紊乱
**解决办法**：
- 壬二酸
- 激光治疗
- 生活方式调整
- 避免诱因
- 定期复查
**预防**：避免刺激

## 6. 斑秃
