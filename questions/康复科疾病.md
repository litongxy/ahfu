# 康复科疾病

## 1. 脑卒中后康复
**症状**：偏瘫、言语障碍、吞咽困难
**病因**：脑卒中后遗症
**解决办法**：
- 运动疗法
- 作业疗法
- 言语治疗
- 吞咽训练
- 物理因子治疗
**预防**：坚持康复

## 2. 脊髓损伤康复
**症状**：肢体瘫痪、感觉障碍、大小便失禁
**病因**：外伤、疾病
**解决办法**：
- 运动训练
- 膀胱训练
- 物理治疗
- 职业训练
- 心理支持
**预防**：避免外伤

## 3. 骨折后康复
**症状**：关节僵硬、肌肉萎缩、功能受限
**病因**：骨折后固定
**解决办法**：
- 关节活动度训练
- 肌力训练
- 物理治疗
- 作业训练
- 循序渐进
**预防**：科学锻炼

## 4. 关节置换术后康复
**症状**：关节疼痛、活动受限
**病因**：关节置换术后
**解决办法**：

- 关节活动度训练
- 肌力训练
- 步态训练
- 预防深静脉血栓
**预防**：规范康复

## 5. 运动损伤康复
**症状**：疼痛、肿胀、活动受限
**病因**：运动相关损伤

**预防**：热身拉伸

## 6. 颈椎病康复
**症状**：颈部疼痛、头晕、上肢麻木
