# 体检报告结论规则来源

本次“体检结论型异常”规则库用于识别体检报告中的常见影像、妇科、眼科、消化、骨科和心电图结论。

说明：

- 规则库的实现文件为 `acp-gateway/src/services/report-special-findings-library.ts`
- 当前已内置 `134` 条常见结论规则
- 这些规则是依据公开的权威健康科普与指南，结合体检报告常见写法整理而成
- 这里的规则用于“文本识别与归类”，不是替代医生诊断的临床标准

## 本次规则覆盖的主要结论类型

- 体重与代谢：BMI增高、超重、肥胖、中心性肥胖、腰围异常、体重偏低
- 甲状腺：结节、囊肿、回声欠均匀、弥漫性病变、肿大、钙化、桥本样改变
- 乳腺：结节、囊肿、增生、小叶增生、纤维腺瘤、导管扩张、钙化
- 妇科：TCT炎症反应性改变、ASC-US、ASC-H、LSIL、HSIL、HPV阳性、子宫肌瘤、卵巢囊肿、盆腔积液等
- 胸部与肺部：肺结节、肺小结节、磨玻璃结节、肺纹理增多、慢支、肺气肿、胸膜增厚、钙化灶、陈旧灶等
- 腹部超声：肝囊肿、肝血管瘤、胆囊息肉、胆囊结石、肾囊肿、肾结石、前列腺增生等
- 心血管与心电图：斑块、IMT增厚、动脉粥样硬化、窦缓、窦速、ST段改变、T波改变、早搏、传导阻滞等
- 消化：慢性胃炎、胃息肉、反流性食管炎、十二指肠炎、幽门螺杆菌阳性、结肠息肉等
- 眼科、耳鼻喉、口腔、骨科：屈光不正、近视、散光、白内障、鼻中隔偏曲、慢性咽炎、牙周炎、骨量减少、骨质疏松、退行性变等
