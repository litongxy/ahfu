# 同仁堂元气助手 - 产品功能文档总结

## 一、总场景梳理

### 核心功能模块

| 场景类型 | 优先级 | 触发来源 | 功能说明 |
|---------|--------|----------|----------|
| 健康咨询 | P0 | 主页聊天输入框 | 用户进行养生、健康常识、疾病防控等基于公共知识库的健康咨询场景 |
| 报告分析 | P1 | 主页聊天输入框/报告上传入口 | 用户上传体检报告后进行分析，获取干预方案 |
| 功能页面路由 | P2 | 主页聊天输入框 | 用户语义中带有APP功能的查询，出现页面快捷入口 |
| 内容推荐 | P4 | 主页聊天输入框 | 用户语义中带有推荐相关内容，出现推荐内容快捷入口 |

## 二、报告分析

### 功能流程

1. **报告上传入口**
   - 用户点击体检报告上传
   - 获取用户体检报告信息并存储

2. **体检结果分析**
   - 结合体检报告数据，判断是否有异常项
   - 检查是否已完成精准健康问卷填写

3. **干预方案输出**
   - 如有异常：更新干预方案，引导用户完成资料填写
   - 根据体检异常结果，给出干预方案建议

4. **AI建议分类**
   - 饮食相关 → 匹配食谱
   - 睡眠及压力相关 → 匹配睡眠和心理课程
   - 体重体脂相关 → 匹配运动课程
   - 其他 → 根据AI语义分析给出建议

### 页面展示

#### 2.1 有异常结果页面
- 显示异常项数量
- 异常项详细分析（如：轻度脂肪肝、ALT轻度升高、尿蛋白微量）
- 每项异常给出复查建议
- 总结及建议：
  - 异常原因分析（如：代谢问题+生活方式相关）
  - 复查时间建议（如：3个月内）
  - 具体复查项目（血压监测、尿常规+肾功能、肝功能、腹部B超）
  - 生活方式调整建议（饮食、运动、作息、体重控制）
- 个性化调理方案推荐：
  - 养生食谱（如：牛肉土豆杂蔬焖菜、玉米鸡胸肉蔬菜焖面，含卡路里）
  - 运动健身（如：HIIT燃脂、健身燃脂课程）
  - 睡眠健康（如：7天轻松入睡、7天正念减压）
- 查看档案入口
- 咨询入口

#### 2.2 无异常结果页面
- 显示"您的体检报告有0项异常"
- 身体健康确认："请继续保持哦~"
- 总结结论：正常报告，建议保持日常健康监测和适当饮食习惯
- 精准健康入口：引导用户完善资料获取全方位健康解决方案

## 总结

本产品为**同仁堂元气助手**，主要包含以下核心能力：

1. **健康咨询** - 基于知识库的养生、健康、疾病防控问答
2. **体检报告分析** - 智能分析体检报告，提供个性化干预方案
3. **内容推荐** - 根据用户需求推荐健康相关图文/视频内容
4. **商品推荐** - 智能推荐相关健康商品
5. **功能导航** - 智能识别用户意图，快速跳转到对应功能页面
6. **后台管理** - 完整的运营管理后台，支持内容/商品/知识库管理等

用户通过**首页聊天输入框**即可触发各类服务，支持体检报告上传、语音/文字咨询等多种交互方式。
